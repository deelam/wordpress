ResultRecord

Retrieve Results

Retrieve ResultRecords by issuing a GET request to the /api/v1/batches/{batchId}/results endpoint. Results will be paginated.

Example request

curl -X GET \
  https://app.alegion.com/api/v1/batches/{batchId}/results \
  -H 'authorization: Bearer {access_token}'

Example response

[
    {
        "id": "046b6c7f-0b8a-43b9-b35d-6489e6daee91",
        "createdAt": "2017-11-14T21:02:36.491Z",
        "resultData": {
[WHAT KINDS OF DATA SHOULD THE CUSTOMER EXPECT?]
        },
        "inputRecord": {
            "createdAt": "2017-11-14T20:59:28.375Z",
            "metadata": "imageset-002",
            "data": {
          "image_url": "http://fileserver.net/photo12.png",
          "image_timestamp": 1563493639
          "image_guid": "CQ4JAQgOCQQDAQgHBAwDDQ"
            },
            "workflowStageId": "046b6c7f-0b8a-43b9-b35d-6489e6daee91",
            "id": "046b6c7f-0b8a-43b9-b35d-6489e6daee91",
            "status": "in-progress"
        }
    }
]

The resultData field contains key-value pairs representing the result of processing the InputRecord through the workflow.

If you populate the metadata or the data fields of the InputRecord, then you should see that same metadata value within the ResultRecord under inputRecord. Additionally the ResultRecord’s id matches that of the InputRecord.

Get Filtered Results

Filter results using the following parameters:

  • query: Full text search
  • minCreatedAt: Minimum creation date filter
  • maxCreatedAt: Maximum creation date filter

Filter request by date

To filter search results by date, use the minCreatedAt and maxCreatedAt query parameters.

curl -X GET \
  https://app.alegion.com/api/v1/batches/{batchId}/results?minCreatedAt=2018-01-01T00:00:00.000Z&maxCreatedAt=2018-01-01T23:59:59.000Z \
  -H 'authorization: Bearer {access_token}'

Filter request using full-text search

To perform a full-text search of records within a batch, use the query parameter. This is often used to search the metadata field to identify a particular record or group of records.

curl -X GET \
  https://app.alegion.com/api/v1/batches/{batchId}/results?query=ABC123 \
  -H 'authorization: Bearer {access_token}'
